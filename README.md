# 📝 Mini Message Board

A simple, elegant message board web application built with Node.js, Express, and EJS. This project was created as part of [The Odin Project](https://www.theodinproject.com/lessons/node-path-nodejs-mini-message-board) curriculum to demonstrate basic Express.js concepts including routing, template rendering, form handling, and data management.

## ✨ Features

- **View Messages** - Browse all messages on the homepage with user details and timestamps
- **Create Messages** - Submit new messages through a user-friendly form
- **Message Details** - Click "View Details" to see individual messages on their own page
- **Responsive Design** - Modern, gradient-themed UI that works on desktop and mobile
- **Real-time Updates** - New messages appear immediately after submission
- **Error Handling** - Graceful handling of missing messages and invalid routes

## 🛠️ Technologies Used

- **Backend**: Node.js, Express.js
- **Template Engine**: EJS (Embedded JavaScript)
- **Frontend**: HTML5, CSS3
- **Development**: Nodemon for auto-reloading
